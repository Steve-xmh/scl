## 功能/特点

- 全异步操作，使用 smol 作为异步框架，快速且轻量
- 全版本启动支持
- 下载纯净游戏
- 下载 Forge 模组安装器
- 下载 Fabric 模组安装器
- 下载 Optifine 模组
- 自定义启动参数
- 正版登录（Mojang, Microsoft（你需要自行获取到回调链接））
- 多下载源（BMCLAPI MC）
- Curseforge 模组检索/下载
