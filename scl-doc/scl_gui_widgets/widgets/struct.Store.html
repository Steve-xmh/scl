pub struct Store<F, T> { /* private fields */ }
Expand description

一个中间件组件，用于拦截数据流并进行处理

和 [druid::Lens] 不一样的地方是，它不强制你去设置数据，这允许你在无法转换数据的情况下保持子代数据不变或父代数据不变。
