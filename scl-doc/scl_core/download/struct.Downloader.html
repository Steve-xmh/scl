Expand description

下载结构，用于存储下载所需的信息，并通过附带的扩展特质下载需要的东西

Fields§

§source: DownloadSource

使用的下载源

§game_independent: bool

是否使用版本独立方式安装

这个会影响 Optifine 以模组形式的安装路径

（会被安装在 版本/mods 文件夹里还是 .minecraft/mods 文件夹里）

§verify_data: bool

是否验证已存在的文件是否正确

§java_path: String

任意的 Java 运行时执行文件目录

在安装 Forge 时会使用

§reporter: Option<R>

下载的进度报告对象
